Dataset Requirements

  • Bibliotecas y Bibliobuses en Madrid

    "Relación de todas la Bibliotecas Públicas sitas en la ciudad de Madrid, de titularidad nacional, comunitaria o local. Incluye también bibliobuses."

    This data is in the smart cities domain because it belongs to the public buildings that the city own.

    The data is available as a CSV file.

    Data have an open license so it can be published.

    The data can be easily linked with generic real-world entities like the libraries, streets and so on.

  • Bibliotecas Especializadas Universitarias y Nacionales

    "Relación de Biliotecas Especializadas Universitarias y Nacionales con indicaciones de horarios y especialidad."

    This data is in the smart cities domain because it belongs to the public and cultural buildings that the city own.

    The data is available as a CSV file.

    Data have an open license so it can be published.

    The data can be easily linked with generic real-world entities like the libraries, streets and so on.

  • Actividades Gratuitas en Bibliotecas Municipales

    "Relación de actividades de carácter gratuito, para público infantil y adulto, desarrolladas por las siguientes instituciones de titularidad municipal: la red de Bibliotecas Públicas, la Biblioteca Histórica, la Biblioteca Musical Víctor Espinós, la Hemeroteca y la Imprenta."

    This data is in the smart cities domain because it belongs to the activities that the city offers.

    The data is available as a CSV file.

    Data have an open license so it can be published.

    The data can be easily linked with generic real-world entities like the libraries, streets and so on.
